# India state boundaries

This GeoJSON is generated by `npm run build:india-map` in the frontend package.

## Sources

| Layer | Source | License |
|-------|--------|---------|
| State/UT polygons | [LGD_States](https://github.com/yashveeeeeeer/india-geodata/releases/tag/admin/states) (Local Government Directory / Survey of India) | CC0 |
| POK merged into J&K | [pok-alhasan.geojson](https://github.com/datameet/maps/tree/master/Country/disputed) (DataMeet composite boundary) | CC BY 4.0 |

The POK polygon is unioned into the Jammu & Kashmir feature so the map matches the official Survey of India political map, which shows the full J&K UT including Pakistan-administered Kashmir.

## Official references

- [Survey of India — Political Map of India](https://surveyofindia.gov.in/pages/political-map-of-india/)
- [Bharat Maps](https://bharatmaps.gov.in/makemymap/makemymap.aspx)

## Regenerating

```bash
cd frontend
npm run build:india-map
```

Requires `7z` on PATH for extracting the LGD release archive.
